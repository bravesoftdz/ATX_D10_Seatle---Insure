Поиск автотранспорта.

Вы можете найти один или несколько транспортных средст используя "Поиск автотранспорта".
Поиск осуществляется по введенному Вами критерию.
Критерий устанавливается следующим образом: если установлена "галочка" напротив одного из атрибутов транспорта, то по этому атрибуту и будет производиться поиск, остальные атрибуты будут игнорироваться.
