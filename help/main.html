Карточки.

При вводе и редактировании карточки автотранспорта, поля помеченные символом (*) не могут быть пустыми.

Для добавления карточки нажмите кнопку "Новая запись".
Для редактирования карточки Вы можете просто изменять значение полей.
Для сохранения карточки нажмите кнопку "Сохранить",
для отмены сделанных изменений - кнопка "Отменить".

"Найти" позволяет быстро перейти к карточки автотранспорта по его Гос. №.
